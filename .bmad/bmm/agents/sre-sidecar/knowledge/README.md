# SRE Agent Knowledge Base

This folder contains domain-specific knowledge for the SRE agent.

## Contents

- `infrastructure-patterns.md` — Common infrastructure patterns and solutions
- `incident-history.md` — Past incidents and learnings

## Usage

The agent will:
1. Reference this knowledge when answering questions
2. Add new patterns and learnings over time
3. Cross-reference current problems with past solutions

## Adding Knowledge

You can add files to this folder to expand the agent's knowledge base:
- Runbooks
- Architecture diagrams
- Common troubleshooting guides
- Team-specific conventions
