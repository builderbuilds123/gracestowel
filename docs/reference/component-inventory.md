# Component Inventory

## Storefront Components (`apps/storefront/app/components`)

### Product Experience
- **`ProductCard.tsx`**: Displays product summary in listings.
- **`ProductDetails.tsx`**: Main product information display.
- **`ProductActions.tsx`**: Add to cart and variant selection logic.
- **`ProductFilters.tsx`**: Filtering sidebar/modal for collections.
- **`ProductImageGallery.tsx`**: Carousel or grid of product images.
- **`ProductInfo.tsx`**: Description and metadata.
- **`ProductPrice.tsx`**: formatting and display of prices.
- **`RelatedProducts.tsx`**: Recommendations widget.
- **`EmbroideryCustomizer.tsx`**: Custom UI for towel personalization.

### Cart & Checkout
- **`CartDrawer.tsx`**: Slide-out cart preview.
- **`CartProgressBar.tsx`**: Step indicator for checkout.
- **`CheckoutForm.tsx`**: Payment and shipping form.
- **`OrderSummary.tsx`**: Order totals and detailed breakdown.
- **`AddItemsDialog.tsx`**: Upsell or quick-add modal.

### Account & User
- **`EditAddressDialog.tsx`**: User address management.
- **`WishlistButton.tsx`**: Toggle for wishlist items.
- **`ReviewForm.tsx`**: Input form for product reviews.
- **`ReviewSection.tsx`**: Display list of user reviews.

### Navigation & Layout
- **`Header.tsx`**: Main site navigation and branding.
- **`Footer.tsx`**: Site links and newsletter signup.
- **`AnnouncementBar.tsx`**: Global banner for promos.
- **`SearchBar.tsx`**: Global search input.

### Shared / Utility
- **`Dropdown.tsx`**: Generic dropdown component.
- **`Map.client.tsx`**: Client-side map rendering (Leaflet).
- **`CountdownTimer.tsx`**: Promotional timer.
- **`ProductDetailSkeleton.tsx`**: Loading state for PDP.

## Design System
- **Styling**: Tailwind CSS v4.
- **Icons**: Phosphor Icons / Lucide React.
